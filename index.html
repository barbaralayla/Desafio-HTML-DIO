Introdução ao HTML

HTML é uma sigla para hypertext markup language, ou linguagem de marcação de hipertexto, então como o nome diz o HTML é uma linguagem de marcação.

O HTML também é conhecido com o esqueleto do site, é nele que é montado a estrutura de um site.

É composto por Tags e Atributos de Tags

  • TAGS: É como se fossem pequenas peças que a gente usa para montar a página. Como um quebra-cabeça.
  • ATRIBUTOS DE TAGS: O que é próprio, característico de algo ou alguém.

Estruturando o HTML

A estrutura base do HTML é composta pelo próprio html, pelo head e pelo body.

  • HTML: Representa a raíz do documento.
  • HEAD: contém informações sobre o documento HTML, como título da página. Essas informações não aparecem para o usuário
  • BODY: Contém o conteúdo visível da página.

No body (corpo do site) existem algumas tags que são tidas como principais e/ou importantes para a estrutura do site. São elas: MAIN, DIV, OL e UL.

  • MAIN: A tag principal, onde vai o conteúdo mais importante do site.
  • DIV: Tag para agrupar e posicionar elemementos.
  • OL: Lista ordenada de itens.
  • UL: Lista desordenada de itens.

Trabalhando com Formulários

O formulário HTML é usado para coletar informações do usuário que são enviadas para processamento em um servidor.

As tags mais comumente usadas para estruturação de um formulário são:

  • FORM: É usado para criar o formulário HTML.
  • INPUT: É o elemento de formulário mais utilizado. Precisa vir acompanhado de um atributo, como:
  • RADIO: Botão de opção que permite marcar somente uma opção entre várias.
  • CHECKBOX: É uma caixa de seleção que permite marcar váriasopções.
  • TEXT: Define um campo de entrada para texto.
  • SUBMIT: Botão para enviar os dados do formulário.
  • BUTTON: Define um botão clicável.
  • SELECT: Define uma lista suspensa.
  • TEXTAREA: Área de texto multilinha.

O cuidado com a acessibilidade é de extrema importância na criação de formulários, para tanto é utilizado a tag label.

  • LABEL: Define um rótulo para vários elementos de formulário. O leitor de tela lerá em voz alta o rótulo.

Trabalhando com Mídias

Mídia na Web é tudo aquilo que é audiovisual, como música, vídeo, foto, animações e pode ser de diferentes tipos e formatos.

As tags comumente utilizadas são:

  • IMG: É usada para incorporar uma imagem a página do site.
  • AUDIO: É usado para reproduzir um arquivo de áudio no site.
  • VIDEO: Usado para mostrar vídeo no site.
  • IFRAME: Serve para incorporar o conteúdo/mídia de uma outra página ao seu site.

Sempre importante utilizar tags e atributos que tornam o site acessível para pessoas com deficiência.

Criando Tabelas com HTML

Uma tabela em HTML consiste em células de tabela dentro de linhas e colunas. As tags mais utilizadas são:

  • TABLE: Tag que define uma tabela.
  • TR: Define uma linha de tabela.
  • TD Define uma célula de tabela.
  • TH Define um cabelhaço de tabela.

Entendendo sobre semântica

Um elemento semântico descreve claramente seu significado tanto para o navegador quanto para o desenvolvedor.

São elementos semânticos as seguintes tags:

  • SECTION: Define uma seção em um documento.
  • ARTICLE: Especifica conteúdo independente e autocontido.
  • HEADER: Elemento representa um contêiner para conteúdo introdutório ou um conjunto de links de navegação
  • FOOTER: Define um rodapé para um documento ou seção.
  • NAV: Define um conjunto de links de navegação.
  • ASIDER: Define algum conteúdo além daquele em que é colocado (como uma barra lateral).
  • FIGURE: Especifica conteúdo independente, como ilustrações, diagramas, fotos, listagens de códigos.
  • FIGCAPTION: Define legenda para a tag figure.
